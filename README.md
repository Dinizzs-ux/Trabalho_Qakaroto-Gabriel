## 💡 Sugestões úteis

* **Alocação Dinâmica:** Garanta que todos os arquivos que utilizam `malloc` façam a verificação de ponteiro nulo para assegurar que a memória foi alocada adequadamente.
* **Ponteiros:** Os desafios de Pilha exigem manipulação direta do topo da estrutura. Se o programa fechar inesperadamente, confira as passagens por referência utilizando o operador `&`.
* **Organização:** Cada atividade está separada logicamente para facilitar a compilação individual e o teste de requisitos específicos.
